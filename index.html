Что это?

Mimimi-tro - это игра про метрополитен и котиков )

Почему?

Вывод:

Это очень интересно из-за необходимости проявлять свои логические и стратегические навыки. А также позволяет оказаться на месте архитектора метрополитена.

И самое главное: игра позволяет весело убить время.

Каковы в ней правила?

Необходимо построть максимально распределенную систему метро, избегая перегрузки какой-либо станции.

Случайно генерируемые элементы

  • Новые станции
  • Пассажиры (котики)

Расходуемые элементы

  • Новые поезда
  • Новые пути

Какова цель игры?

Набирать как можно больше очков и бить рекорды.

А выиграть в ней нельзя!

Техническая часть

Backend

  • Java
  • Spring framework
  • Maven
  • PostgreSQL

Frontend

  • Java Script
